## 快速开始 - 新项目

您可以通过从 GitHub 克隆仓库来开始一个全新的项目。在终端中运行以下命令：

```bash [Terminal]
git clone https://github.com/zxml7777777/fork-nuxt-SaaS-starter
```

→ 进入项目目录：

```bash [Terminal]
cd nuxt-saas-template
```

→ 安装依赖：

```bash [Terminal]
pnpm install
```

→ 在根目录创建一个 `.env` 文件或复制/粘贴 `.env.example` 文件

### 配置

在使用 `pnpm run dev` 之前，让我们检查配置部分以更新所有环境变量
